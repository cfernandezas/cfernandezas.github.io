Intención de diseño

Creación de puestos de feria utilizando el workshop de manera que los mismos feriantes puedan construirlo. Conceptos claves:

  • Materiales económicos
  • Fácil ensamblaje y construcción
  • Luminosidad
  • Techo apto para posteriores paneles solares
  • Mismo estilo con el resto del parque

Objetivo

Crear un espacio de intercambio cultural para el desarrollo endógeno a fin de potenciar las capacidades internas de la comunda de Lo Prado y alrededores. Con esto se pretende impuslar la economía de las personas que viven en el sector.
